## Useful Commands

| Command         | Purpose             |
| --------------- | ------------------- |
| `go run .`      | Run current project |
| `go build`      | Build binary        |
| `go test ./...` | Run all tests       |
| `go fmt ./...`  | Format code         |
| `go vet ./...`  | Static analysis     |
| `go mod tidy`   | Clean dependencies  |
